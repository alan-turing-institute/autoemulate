20 dataset: str

22 - **cardiac1**: ionic atrial cell model data from LH sampling.

34 - **engineering1**: Cantilever truss simulation. 

35 - **reactiondiffusion1**: Reaction-diffusion system simulated data. Spatial 

36 snapshots in correspondence for different values of reaction 

37 and ddiffusion parameters. (50 snapshots in correspondence of 

38 50 different parameter sets at time T=10). 

39 split: bool 

40 Whether to split the data into training and testing sets. Default is False. 

41 test_size: float 

42 The proportion of the dataset to include in the test split. Default is 0.2. 

43 random_state: int 

44 Controls the shuffling applied to the data before applying the split. Default 

45 is 42. 

46 

47 Returns 

48 ------- 

49 list[TensorLike] 

50 Tensors of simulation parameters / inputs and simulation outputs. If `split` 

51 is True, returns [x_train, x_test, y_train, y_test]. 

52 """
